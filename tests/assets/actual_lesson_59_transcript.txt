<p>Kirin: <span    color=""   simp="这个" trad="這個" py="Zhège" eng="This one" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">这个</span>.</p>
<p>Adam: Very cute.</p>
<p>Kirin: <span    color=""   simp="很可爱" trad="很可愛" py="Hěn kěài" eng="Very cute" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">很可爱</span>.</p>
<p>Adam: Clothing or clothes.</p>
<p>Kirin: <span    color=""   simp="衣服" trad="衣服" py="Yīfú" eng="Clothes" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">衣服</span>.</p>
<p>Adam: That coat is very cute.</p>
<p>Kirin: <span    color=""   simp="那个外套很可爱" trad="那個外套很可愛" py="Nàge wàitào hěn kěài" eng="That coat is very cute" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">那个外套很可爱</span>.</p>
<p>Adam: Would you like to buy it?</p>
<p>Kirin: <span    color=""   simp="你想要买吗" trad="你想要買嗎" py="Nǐ xiǎng yào mǎi ma" eng="Would you like to buy it" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">你想要买吗</span>?</p>
<p>Adam: “I don’t think so.”  Or literally “Don’t think so.”</p>
<p>Kirin: <span    color=""   simp="不想" trad="不想" py="Bù xiǎng" eng="I don&#8217;t think so" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">不想</span>.</p>
<p>Adam: It’s too expensive.</p>
<p>Kirin: <span    color=""   simp="太贵了" trad="太貴了" py="Tài guì le" eng="Too expensive" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">太贵了</span>.</p>
<p>Adam: It’s very cheap.</p>
<p>Kirin: <span    color=""   simp="很便宜" trad="很便宜" py="Hěn piányí" eng="Very cheap" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">很便宜</span>.</p>
<p>Adam: How much is this one?</p>
<p>Kirin: <span    color=""   simp="这个多少钱" trad="這個多少錢" py="Zhège duō shǎo qián" eng="How much is this one" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">这个多少钱</span>?</p>
<p>Adam: This one is 563 dollars.  Use <span    color=""   simp="元" trad="元" py="yuán" eng="dollars" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">元</span>.</p>
<p>Kirin: <span    color=""   simp="这个五百六十三元" trad="這個五百六十三元" py="Zhège wǔ bǎi liù shí sān yuán" eng="This one is 563 dollars" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">这个五百六十三元</span>.</p>
<p>Adam: That’s right.</p>
<p>Kirin: <span    color=""   simp="对" trad="對" py="Duì" eng="That&#8217;s right" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">对</span>.</p>
<p>Adam: You should find.</p>
<p>Kirin: <span    color=""   simp="你应该要找" trad="你應該要找" py="Nǐ yīnggāi yào zhǎo" eng="You should find" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">你应该要找</span>.</p>
<p>Adam: A little bigger.</p>
<p>Kirin: <span    color=""   simp="比较大一点的" trad="比較大一點的" py="Bǐjiào dà yīdiǎn de" eng="A little bigger" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">比较大一点的</span>.</p>
<p>Adam: A little smaller.</p>
<p>Kirin: <span    color=""   simp="比较小一点的" trad="比較小一點的" py="Bǐjiào xiǎo yīdiǎn de" eng="A little smaller" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">比较小一点的</span>.</p>
<p>Adam: In that case.</p>
<p>Kirin: <span    color=""   simp="那么" trad="那麼" py="Nàme" eng="In that case" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">那么</span>.</p>
<p>Adam: How about that one?</p>
<p>Kirin: <span    color=""   simp="那个呢" trad="那個呢" py="Nàge ne" eng="How about that one" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">那个呢</span>?</p>
<p>Adam: I think it looks great.</p>
<p>Kirin: <span    color=""   simp="我觉得很好看" trad="我覺得很好看" py="Wǒ juéde hěn hǎo kàn" eng="I think it looks great" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">我觉得很好看</span>.</p>
<p>Adam: Why?</p>
<p>Kirin: <span    color=""   simp="为什么" trad="為什麼" py="Wèi shénme" eng="Why" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">为什么</span>?</p>
<p>Adam: Because.</p>
<p>Kirin: <span    color=""   simp="因为" trad="因為" py="Yīnwèi" eng="Because" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">因为</span>.</p>
<p>Adam: I really like this color.</p>
<p>Kirin: <span    color=""   simp="我很喜欢这个颜色" trad="我很喜歡這個顏色" py="Wǒ hěn xǐhuān zhège yánsè" eng="I really like this color" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">我很喜欢这个颜色</span>.</p>
<p>Adam: So, hopefully, you were able to get most of those.  Let’s listen to this dialogue one more time.</p>
<p>Cindy: <span    color=""   simp="这个衣服很可爱" trad="這個衣服很可愛" py="Zhè ge yīfú hěn kěài" eng="These clothes are very cute" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">这个衣服很可爱</span>.<br />
Yann: <span    color=""   simp="你想要买吗" trad="你想要買嗎" py="Nǐ xiǎng yào mǎi ma" eng="Would you like to buy them" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">你想要买吗</span>?<br />
Cindy:  <span    color=""   simp="不想" trad="不想" py="Bù xiǎng" eng="I don&#8217;t think so" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">不想</span>.  <span    color=""   simp="太贵了" trad="太貴了" py="Tài guì le" eng="It&#8217;s too expensive" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">太贵了</span>!<br />
Yann: <span    color=""   simp="那个多少钱" trad="那個多少錢" py="Nàge duō shǎo qián" eng="How much is it" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">那个多少钱</span>?<br />
Cindy: <span    color=""   simp="这个一千三百块" trad="這個一千三百塊" py="Zhège yī qiān sān bǎi kuài" eng="This one is $1300 dollars" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">这个一千三百块</span>.<br />
Yann: <span    color=""   simp="对，太贵了" trad="對，太貴了" py="Duì, tài guì le" eng="You&#8217;re right, it&#8217;s too expensive" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">对，太贵了</span>.  <span    color=""   simp="你应该要找比较便宜一点的" trad="你應該要找比較便宜一點的" py="Nǐ yīnggāi yào zhǎo bǐjiào piányí yīdiǎn de" eng="You should find something a little cheaper" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">你应该要找比较便宜一点的</span>.<br />
Cindy: <span    color=""   simp="那么，这个裤子呢" trad="那麼，這個褲子呢" py="Nàme, zhège kùzi ne" eng="How about these pants" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">那么，这个裤子呢</span>?<br />
Yann: <span    color=""   simp="我觉得不好看" trad="我覺得不好看" py="Wǒ juéde bù hǎo kàn" eng="I don&#8217;t think they look very good" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">我觉得不好看</span>.<br />
Cindy: <span    color=""   simp="为什么" trad="為什麼" py="Wèi shénme" eng="Why not" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">为什么</span>?<br />
Yann: <span    color=""   simp="因为我不喜欢它的颜色" trad="因為我不喜歡它的顏色" py="Yīnwèi wǒ bù xǐhuān tāde yánsè" eng="Because I don&#8217;t like their color" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">因为我不喜欢它的颜色</span>.</p>
<p>Adam: So there we have it, a quick review of some of the vocabulary we learned in unit one.  We have one more review lesson like this planned before we wrap up Unit One.  And as always, for more translation and review tools, consult the Premium section of our website ChineseLearnOnline.com.</p>
<p>Kirin: <span    color=""   simp="再见" trad="再見" py="Zàijiàn" eng="Goodbye" class="has-tip_removed lesson_language lesson_translate lesson_translate_parts ">再见</span>.</p>
